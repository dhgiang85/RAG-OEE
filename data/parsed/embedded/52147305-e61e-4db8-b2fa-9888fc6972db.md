# Manual source-other test

Đây là file Markdown viết tay để test Workstream B (Phase 2 embedder).
File này KHÔNG đi qua parser-catalog. Embedder sẽ:

1. Đọc front-matter → detect `source: other` + `doc_id` chưa có
2. Compute `sha256(body)` rồi dedupe-check
3. INSERT row vào `documents` với `source='other'`, `parser_engine=NULL`
4. Rewrite front-matter — thêm `doc_id`, `origin.size_bytes`, `origin.sha256`
5. Rename file thành `<doc_id>.md`
6. Chunk + embed body
7. Move sang `data/parsed/embedded/<doc_id>.md`

## Phần thử nghiệm

Đoạn văn này đủ dài để chunker tạo ít nhất một chunk. Người dùng có thể
soạn nội dung tùy ý — embedder chỉ quan tâm tới việc:
- Body có đủ token để qua filter `is_degenerate_chunk`
- Front-matter hợp lệ theo `shared/schemas/front-matter.schema.json`

## Kết quả mong đợi

Sau khi drop file này vào `data/parsed/accepted/`:

| Kiểm tra | Kết quả mong đợi |
|---|---|
| File ở `data/parsed/accepted/` | biến mất (đã rename + move) |
| File ở `data/parsed/embedded/<doc_id>.md` | tồn tại |
| Row trong `documents` | `source=other`, `parser_engine IS NULL`, `status=ready` |
| Row trong `chunks` | ≥ 1 chunk với `vector_dims(embedding) = 1024` |
